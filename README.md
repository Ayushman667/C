# C Learning Journey

A collection of C programming projects created while learning the C programming language and fundamental software development concepts.

This repository documents my progress from basic programming concepts to larger, menu-driven applications involving dynamic memory allocation, structures, file handling, pointers, and data management.

## About

The goal of this repository is to:

* Practice core C programming concepts
* Build increasingly complex projects
* Improve problem-solving skills
* Learn memory management and data structures
* Develop software engineering habits such as version control and documentation

Most projects are written as standalone console applications and focus on implementing real-world functionality using only standard C.

## Projects

## Compilation

Most projects can be compiled using GCC:

```bash
gcc filename.c -o output
```

Example:

```bash
gcc contact_manager.c -o contact_manager
./contact_manager
```
